 Create or Access Your Microsoft 365 Tenant
Navigate to Microsoft 365 Admin Center.
Sign in with your global administrator account or create a new account if this is your first setup.
If setting up a new tenant:
Enter your organization name, region, and domain name.
Microsoft will guide you through the initial subscription link and verification of domain ownership.
 Create Users and Assign Licenses
In Admin Center, navigate to Users → Active Users → Add a user.
Fill in user details:
First Name / Last Name
Username (email)
Password (auto-generate or custom; recommend enforcing password change on first login)
Assign license:
Choose the appropriate Microsoft 365 plan (Business Basic, Standard, Premium, or E3/E5).
Optionally enable or disable individual services like Outlook, Teams, SharePoint, OneDrive.
Assign roles if necessary:
Global Admin for administrative access.
Regular User for standard accounts.
